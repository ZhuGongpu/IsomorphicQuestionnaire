# Redux Web Template

`React` + `React-Router` + `Redux` + `Webpack` + `Sass` boilerplate.

## Features

- Hot reloading.
- Code splitting by react-router and webpack.
- Predictable state mutations.
- Sass and PostCSS support.
- CSS modules.

## How to use

1. Clone this repo.
2. `npm install`.
3. `npm start`.
4. Happy hacking!
